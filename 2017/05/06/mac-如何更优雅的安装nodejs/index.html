发表于 2017-05-06 |

用了n觉得并不好用，后来才知道mac有nvm，简直方便的一米啊。

安装nrm

nrm 是一个管理 npm 源的工具。用过 ruby 和 gem 的同学会比较熟悉，
通常我们会把 gem 源切到国内的淘宝镜像，这样在安装和更新一些包的时候比较快。
nrm 同理，用来切换官方 npm 源和国内的 npm 源（如: cnpm），当然也可以用来切换官方 npm 源和公司私有 npm 源。
